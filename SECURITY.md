# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.x | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do **not** open a public issue.

Instead, please use one of these channels:

- GitHub Security Advisories / private vulnerability reporting for this repository
- Email: `eibrahimov@gmail.com`

When reporting, please include:

- a description of the issue
- reproduction steps or a proof of concept
- affected versions
- any suggested remediation, if available

I will acknowledge receipt as quickly as possible and aim to provide an initial response within 3 business days.

## Disclosure Process

- Please allow time for investigation and remediation before public disclosure.
- Once a fix is available, a patched release will be published and the advisory will be updated with impact and remediation details.
